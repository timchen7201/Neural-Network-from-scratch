# Neural Network from scratch
Building a neural network by `numpy` only

* Dataset
    * Mnist
* Model Summary
    * Input Layer: 768
    * Hidden Layer: 128
    * Output Layer: 10
## Training
* Install package
```bash
$ pip install -r requirements.txt
```
and then
```bash
$ python main.py
```
